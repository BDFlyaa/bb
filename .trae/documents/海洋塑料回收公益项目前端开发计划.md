# 海洋塑料回收公益项目前端开发计划

## 一、项目初始化与依赖安装

1. **安装核心依赖**

   * Pinia (状态管理)

   * Vue Router (路由管理)

   * TailwindCSS + Headless UI (样式与组件库)

   * Three.js + troika-three-text + GSAP (3D与动画效果)

   * ECharts (数据可视化)

   * 高德地图 JS API 2.0 (地图功能)

2. **配置项目**

   * 配置 TailwindCSS

   * 配置 Vue Router

   * 配置 Pinia

   * 配置环境变量（高德地图API密钥等）

## 二、项目结构设计

```
src/
├── assets/          # 静态资源
│   ├── images/      # 图片资源
│   ├── videos/      # 视频资源
│   └── styles/      # 全局样式
├── components/      # 通用组件
│   ├── common/      # 基础通用组件
│   ├── ocean/       # 海洋特色组件
│   └── charts/      # 图表组件
├── composables/     # 组合式函数
├── layouts/         # 布局组件
├── pages/           # 页面组件
│   ├── auth/        # 认证相关页面
│   ├── map/         # 地图相关页面
│   ├── checkin/     # 打卡相关页面
│   ├── blockchain/  # 区块链溯源页面
│   ├── stats/       # 统计与可视化页面
│   ├── mall/        # 积分商城页面
│   ├── science/     # 科普专区页面
│   └── community/   # 社区页面
├── router/          # 路由配置
├── stores/          # Pinia 状态管理
├── types/           # TypeScript 类型定义
├── utils/           # 工具函数
├── App.vue          # 根组件
└── main.ts          # 入口文件
```

## 三、各模块实现计划

### 1. 用户与权限管理模块

* 实现登录/注册页面

* 实现角色权限控制

* 实现用户信息管理

* 集成高德地图API

* 实现回收点信息申报表单

* 实现回收点审核功能

### 3. 志愿者打卡模块

* 实现拍照上传功能

* 集成AI塑料垃圾识别（模拟实现）

* 实现地理位置校验

### 4. 回收数据区块链溯源模块

### 7. 海洋塑料知识科普专区

* 实现图文展示

* 实现互动H5

### 8. 志愿者社区模块

* 实现任务广场

* 实现组队捡拾功能

* 实现动态发布与评论点赞

### 9. 海洋特色UI动效

* 全局水波纹效果

## 五、海洋特色UI设计方案

1. **配色方案**

   * 主色调：深海蓝 #0F172A

   * 辅助色：海绿色 #10B981、珊瑚红 #EF4444

   * 中性色：浅蓝灰 #F1F5F9、深蓝灰 #1E293B

2. **动画效果**

   * 使用 GSAP 实现流畅的过渡动画

   * 使用 Three.js 实现 3D 海洋场景

   * 实现塑料瓶漂浮粒子效果

   * 设计贝壳形状的加载动画

3. **交互设计**

   * 水波纹点击效果

   * 平滑滚动过渡

   * 悬浮卡片效果

   * 动态导航栏

## 六、技术亮点

4. **模块化开发**：便于维护和扩展
5. **海洋特色UI**：打造独特的视觉风格
